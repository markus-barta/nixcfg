## Manual Test Procedures

### Test 1: Starship Prompt Theme

**Steps:**

1. Open a new terminal (SSH or local)
2. Verify the prompt has **Yellow** gradient colors
3. Check the powerline segments are visible

**Expected Results:**

- Prompt displays with Yellow gradient colors
- OS icon visible on the left
- Directory path visible
- Git branch (if in a git repo) visible with muted count
- Time on the right side
- "nix" indicator when in nix-shell

**Status:** ⏳ Pending

### Test 2: Starship Config File

**Steps:**

1. Check config file exists:
   ```bash
   cat ~/.config/starship.toml | head -15
   ```

**Expected Results:**

- Header shows "hsb0 (Yellow)"
- Theme: Yellow (home)
- Palette Key: yellow
- Auto-generated by: modules/shared/theme-hm.nix

**Status:** ⏳ Pending

### Test 3: Root Alert (Security Feature)

**Steps:**

1. Switch to root: `sudo -s`
2. Observe the prompt

**Expected Results:**

- ⚠ symbol appears at far left with red background
- Indicates privileged shell session

**Status:** ⏳ Pending

### Test 4: Command Duration

**Steps:**

1. Run a slow command: `sleep 3`
2. Check if duration appears

**Expected Results:**

- Duration shows (e.g., "⏱ 3s") after command completes

**Status:** ⏳ Pending

### Test 5: Error Status

**Steps:**

1. Run a failing command: `false`
2. Check the prompt character

**Expected Results:**

- Prompt character changes to ✗ in red

**Status:** ⏳ Pending

### Test 6: Zellij Theme

**Steps:**

1. Start zellij: `zellij`
2. Check the status bar colors

**Expected Results:**

- Status bar uses Yellow theme colors

**Status:** ⏳ Pending

### Test 7: Eza Colors
