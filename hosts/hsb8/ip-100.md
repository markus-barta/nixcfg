# 192.168.1.100 — hsb8 (Parents' Home Server)

**Role:** Home automation + DNS/DHCP server at parents' home (ww87 location)

| Property        | Value                       |
| --------------- | --------------------------- |
| IP Address      | `192.168.1.100/24`          |
| Interface       | `enp2s0f0`                  |
| Default Gateway | `192.168.1.1` (router)      |
| DNS Server      | `127.0.0.1` (local AdGuard) |
| Search Domain   | `local`                     |

## AdGuard Home (DNS + DHCP)

- **Web Interface:** <http://192.168.1.100:3000>
- **DHCP Range:** 192.168.1.225 → 192.168.1.254 (statics own `.2–.221`, `.222–.224` reserved as buffer)
- **Upstream DNS:** Cloudflare (1.1.1.1, 1.0.0.1)
